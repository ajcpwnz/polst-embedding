# polst-embedding

Sandbox apps for embedding the Polst SDKs.

- `ios/` contains the iOS sandbox app.
- `android/` contains the Android sandbox app.
- `docs/` contains the public integrator demo site, deployed via GitHub Pages.

See `ios/README.md` for iOS run instructions.

## Demo site

Live integrator demo: <https://ajcpwnz.github.io/polst-embedding/>

The demo lists every embedding mode (iframe, script tag, JS SDK, REST,
iOS, Android, React Native). Source lives in `docs/` and deploys
automatically from `master` via GitHub Pages.
